## Steps to start server

#### 1. Clone repository

```
git clone https://github.com/IshaanOhri/Spreen
```

#### 2. Start Spreen server

- #### Method 1 - Using NPM Script

#### Install packages

```
npm ci
```

#### Start server:

```
npm start
```

- #### Method 2 - Using Docker

#### Start docker container:

```
docker-compose up
```

#### 3. Open browser:

- Start screen share

```
http://localhost:3000
```

- View screen share

```
http://localhost:3000/watch
```
